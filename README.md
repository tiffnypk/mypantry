# mypantry

mypantry is a web application that generates personalized recipe suggestions based on the ingredients a user already has at home. Users enter what's in their pantry, fridge, or freezer; mypantry returns a ranked set of recipes they can actually make right now, filtered by dietary needs and saved to their personal cookbook for future reference.
The application exists to remove the daily friction of "what do I cook tonight?" — a question that drives food waste, takeout spend, and decision fatigue for tens of millions of households.

## Features

- **Generate recipes** from ingredients you have on hand
- **Dietary filters** — vegetarian, vegan, gluten-free, and more
- **Pantry management** — save your go-to ingredients so they're always pre-loaded
- **Favorites** — heart any recipe to save it for later
