# mypantry

mypantry is a web application that generates personalized recipe suggestions based on the ingredients a user already has at home. users enter what's in their pantry, fridge, or freezer; mypantry returns a ranked set of recipes they can actually make right now, filtered by dietary needs and saved to their personal cookbook for future reference.<br><br>
the application exists to remove the daily friction of "what do I cook tonight?" — a question that drives food waste, takeout spend, and decision fatigue for tens of millions of households.<br>

## features

- **generate recipes** from ingredients you have on hand
- **dietary filters** — vegetarian, vegan, gluten-free, and more
- **pantry management** — save your go-to ingredients so they're always pre-loaded
- **favorites** — heart any recipe to save it for later
